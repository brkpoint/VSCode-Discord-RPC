### 1.0.0 VSCode-RPC
This is an extension to vscode for discord rpc

## Features
- Working on, Shows in the RPC what are you currently working on
- Language icon display, displays the language icon in the RPC
- Time elapsed, self explanatory

## Requirements
Discord installed
